# voice-editing-app

### Cut, Join, Noise Removal, Enhance to studio level.

## 🚀 Overview

Voice Editing App is a powerful tool that allows users to **Cut and join different audios**, **remove background noise**, and **enhance to studio level sound quality** for a seamless audio experience. Ideal for content creators, podcasts solutions, and ...

## 🌟 Features

✅ **Cut** - Cut any part of audio, delete any part.

✅ **Join** - Join 2 audios.

✅ **Noise Removal** - Filters out background noise for crystal-clear studio level audio.

🛠️ Technologies Used

- **Frontend:** React.js
- **Backend:** Python (Flask/FastAPI)
- **AI & Processing:**
  - dhwanim_v0 - our proprietery model
  - ffmpeg, Librosa & Pydub for audio processing
